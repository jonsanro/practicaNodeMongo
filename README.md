# PracticaNodeApi

## Install

```shell
npm install
```

Copy .env.example to .env and review the values.

## Database

To initialize the database use:

```shell
npm run install_db
```

## Run

To start the application in production use:

```shell
npm start
```


## Development

To run the application in development use:

```shell
npm run dev
```
